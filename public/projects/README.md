# Project Images Folder

Letakkan foto-foto project Anda di folder ini dengan nama file yang sesuai:

## Daftar Gambar yang Dibutuhkan:

1. **sidesa-platform.jpg** - Screenshot/thumbnail SiDesa Platform
2. **laravel-ecommerce.jpg** - Screenshot/thumbnail Laravel E-Commerce (Toko Sepatu)
3. **online-course-platform.jpg** - Screenshot/thumbnail Online Course Platform
4. **zombieknight-game.jpg** - Screenshot/thumbnail ZombieKnight Video Game

## Format File:
- Format: JPG, JPEG, atau PNG
- Rekomendasi ukuran: 800x600 px atau 1200x800 px (landscape)
- Resolusi: minimal 72 DPI
- Ukuran file: maksimal 500KB untuk performa optimal

## Tips Screenshot:
- Ambil screenshot dari halaman utama/landing page project
- Pastikan tampilan bersih dan profesional
- Untuk game, gunakan gameplay screenshot atau cover art
- Crop dan resize sesuai kebutuhan sebelum upload

## Cara Update:
1. Siapkan foto project Anda
2. Rename sesuai dengan nama file di atas
3. Copy ke folder `public/projects/`
4. Refresh browser untuk melihat perubahan
